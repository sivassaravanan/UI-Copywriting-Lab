# UX Lab - Sign-Up Platform

A clean, conversion-focused user interface design for **UX Lab**, a platform offering high-conversion microcopy patterns. 

Refer to the design file named **"UI Copywriting Lab.png"** for the visual layout reference.

---

##  Design Features

*   **Header & Footer:** Clean top branding with a global navigation footer featuring `Home`, `Workbench`, `Library`, and `Profile` (active state).
*   **Value Proposition:** Clear header ("Join the Laboratory") highlighting the core benefit of 100+ microcopy patterns.
*   **Themed CTA:** High-contrast emerald green primary button reading **"Start Your Editorial Journey"**.
*   **Authentication Options:** Standard email/password form alongside a "Sign up with Google" alternative login route.

---

##  Design System

*   **Typography:** Serif headings for an editorial feel, paired with a highly legible Sans-Serif font for form inputs and tabs.
*   **Primary Color:** Emerald Green (`#006644`) used for key CTAs, branding accents, and active navigation states.
*   **UX Details:** Contextual form placeholders (e.g., `E.g. Elena Rossi`) and an integrated password visibility toggle to reduce user friction.
